# Product Compare AI

## 🚀 Overview

A personal decision system for product evaluation.

This project is not just a product logging app.
It is designed to help users **collect, compare, and evaluate products** based on their own judgment.

The ultimate goal is to evolve into an
👉 **image-based product search and decision engine**

---

## 🎯 Core Concept

Most apps store data.

This system stores **decisions**.

* Data → stored
* Decisions → accumulated
* Patterns → discovered

---

## 🧠 Key Features

### 1. Product Registration

* Multiple images per product
* Brand, price, link
* Personal evaluation scores

  * Design
  * Usability
  * Durability
  * Price satisfaction

---

### 2. Product Comparison

* Compare 2~4 products side by side
* Score-based comparison
* Decision tracking (good / bad / hold / purchased)

---

### 3. Camera-based Search (Planned)

* Capture product image
* Compare with stored product database
* Return similar products
* Show "No Match" if none found

---

## 🏗️ Architecture

Frontend → Backend → AI Engine → Database

* Frontend: React
* Backend: Node.js
* AI: Python (CLIP-based image embedding)
* Database: PostgreSQL + pgvector
